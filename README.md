# 🐝 獨居蜂的秘密生活

一份適合**國小中高年級**的自然科互動數位教材。帶學生認識「獨居蜂」這群不住蜂窩、自己築巢、個性溫和的授粉小幫手。

線上版（部署後）：`https://prayer168.github.io/Solitary-Bees/`

## 教材簡介

大部分人以為蜜蜂都住在大蜂窩裡，其實全世界超過 9 成的蜂類都是「獨居蜂」。
本教材用科技深色互動風格，透過比較、滑桿模擬、點擊任務與闖關測驗，
讓學生認識獨居蜂的生活方式，並學會和牠們和平相處。

## 適用對象

- 國小中年級、高年級自然科
- 教學時間：約 40～80 分鐘（課堂導入、分組操作或學生自學皆可）
- 裝置：電腦、平板、手機、觸控螢幕、投影機（RWD 響應式設計）

## 教材內容（6 個分頁）

| 分頁 | 內容 | 互動 |
|---|---|---|
| 🏠 開始 | 首頁主視覺、SVG 飛舞的蜜蜂與花朵 | 開始按鈕 |
| 🎯 學習目標 | 三大學習目標卡 + 教材導覽 | — |
| 🔍 認識獨居蜂 | 獨居蜂 vs 社會性蜂 比較表 | 對照閱讀 |
| 🏗️ 築巢觀察 | 滑桿模擬母蜂一步步築巢 | 滑桿 + 即時動畫 + 數據 |
| 🏨 蓋昆蟲旅館 | 挑選材料蓋一間昆蟲旅館 | 點選 + 即時回饋 |

## 使用方式

1. **直接開啟**：用瀏覽器打開 `index.html` 即可（純前端、可離線）。
2. **本機預覽**（建議，避免某些瀏覽器限制）：
   ```bash
   python -m http.server 8200
   # 開啟 http://localhost:8200
   ```
3. **部署到 GitHub Pages**：見下方「部署」。

## 檔案結構

```text
Solitary-Bees/
├── index.html        # 主教材網頁（6 分頁 SPA）
├── style.css         # 科技深色 glassmorphism 設計系統
├── script.js         # 分頁切換、滑桿模擬、昆蟲旅館、測驗、證書
├── README.md         # 本說明文件
├── teacher-guide.md  # 教師教學指引
└── .claude/
    └── launch.json   # 本機預覽伺服器設定
```

## 部署到 GitHub Pages

```bash
git add -A
git commit -m "獨居蜂互動教材"
git push -u origin main
```

到 GitHub 倉庫 **Settings → Pages**，將來源設為 `main` 分支根目錄，
約 1～2 分鐘後即可於 `https://prayer168.github.io/Solitary-Bees/` 看到。

> 提醒：首頁檔名為 `index.html`、所有資源使用相對路徑，符合 GitHub Pages 需求。

## 教師使用建議

- **投影導入**：用「🏠 開始」與「🔍 認識獨居蜂」帶全班討論。
- **分組操作**：讓學生在平板上玩「🏗️ 築巢觀察」與「🏨 蓋昆蟲旅館」。
- **學習檢核**：以「🏆 闖關大挑戰」收尾，學生可列印自己的結業證書。
- 詳細課程流程與討論問題見 `teacher-guide.md`。
